# Project

Telegram bot for song lyrics risk analysis.

Purpose:
Analyze song lyrics and identify potentially risky content.

Categories:

- Drug references
- Drug glorification
- Criminal activity
- Violence
- Sexual content
- Non-traditional relationships
- Extremist references

Risk levels:

GREEN
YELLOW
RED

Requirements:

- Telegram bot
- Python
- aiogram 3
- SQLite
- Logging
- Docker support

Output:

- Overall risk
- Risk score
- Suspicious fragments
- Explanation
- Literal meaning
- Metaphorical meaning
- Confidence

Additional:

Generate replacement idea clouds instead of rewriting lyrics.
